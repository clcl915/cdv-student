Cafes I've been to in Shanghai

how to read

size of cafe (tiny,smol,med,lg)-> size of coffee bean

Located in which district --> bean roast

  • Channing -> beige
  • Huangpu -> earth yellow
  • Jingan -> dark orange
  • Pudong-> red brown
  • Putuo -> gray
  • Xuhui -> brown

good seats for working? (no,meh,yes) --> aroma waves (none, 1, 2)

anything special? -> right symbols

  • Dough -> *
  • Cakes -> #
  • BAGELS! -> o
  • Animals -> +
  • Paw -> !
